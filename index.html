Service

サービス

SEO / Marketing

SEOでターゲットにしたいキーワードの選定から記事作成、運用まで対応いたします。キーワードの選定では検索ボリューム（検索数）の調査や競合分析を行い、レポートで可視化して報告します。その上で、キーワードの選定や、選定したキーワードでの上位表示のための施策立案、実行をご提案します。

Design

ウェブサイトやバナーのデザインをします。新規サイトの作成やウェブサイトのリニューアルはもちろん、「既存サイトの一部だけデザインを変更する」やバナーの修正など小回りの効く対応が可能です。ウェブサイトのデザインと合わせたパンフレットや名刺、ノベルティの作成なども対応可能です。

Coding / Programming

デザインデータの通りにウェブサイトを構築したり、アニメーションを実装したり、送信フォームやシステムの開発、条件分岐を用いた表示ロジックの調整など対応します。様々な画面サイズで表示が崩れないように調整するレスポンシブ対応が可能です。（PC、スマホ、タブレット対応）。

Wordpress / CMS

ワードプレスなどCMS（コンテンツマネジメントシステム）をウェブサイトに導入する対応ができます。CMSを使うことで、ITエンジニアでなくても管理画面から簡単にウェブサイトの更新ができるようになります。自由度の高いカスタマイズができる技術があるので、運用しやすいサイトの構築が可能です。

SNS / Marketing

Facebook、Instagram、Twitter、You TubeなどSNSに埋め込みしたり、ウェブサイトと連動させたSNSの運用を代行したりすることでより商品・サービスへのリーチが増やす対応が可能です。SNSの広告運用やアクセス解析、投稿用の画像加工や動画編集など多岐にわたるサポートが可能です。
